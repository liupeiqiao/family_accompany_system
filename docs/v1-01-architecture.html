LLM 上下文注入（每轮对话）

回复生成前，以下信息全部注入 System Prompt：

👴 老人画像

ElderProfile — 姓名·性别·性格·喜好·习惯·健康·说话特点·经历

👤 当前角色画像

PersonaProfile — AI扮演谁·称呼·性格·说话风格·陪伴方式

👥 提及人物信息

PersonaProfile + FamilyProfile — 被提到角色的画像 + 家人偏好档案 + 关系网

📝 Top-5 记忆

MemoryUnit ×5 — 评分排序后的最佳记忆·含subject·情感标签

🎯 共情策略

select_strategy() — 指导回复基调（分享/安慰/转移/鼓励…）

🔗 家人关系表

FamilyProfile.relations — 家人之间关系（如"刘承文是张怀粉的丈夫"）
