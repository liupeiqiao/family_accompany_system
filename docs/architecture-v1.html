评分引擎：Score = αR + βE + γC + δS − εM

R — 相关性

话题匹配 (40%)Jaccard系数
家人匹配 (25%)提及检测
时间衰减 (15%)e⁻ˡᵃᵐᵇᵈᵃ·ᵗ
语义补丁 (20%)关键词重叠

E — 共情度

情绪匹配 (40%)标签一致性
历史有效 (35%)策略效果反馈
冲突矩阵 (25%)56格三级判定 ✅⚠️❌

C — 亲密度

关系权重 (50%)配偶1.0/子女0.9/...
记忆权重 (30%)intimacy_weight
访问热度 (20%)调用频次

S — 安全性 + M 罚分

风险词检测病/死/钱/遗产
医学边界诊断/药物建议 → S=0
敏感话题已故亲人 → M=0.5
夜间保护21:00-06:00 遗憾标签 ❌
