03 — 核心算法设计

评分引擎 · 情感冲突矩阵 · 策略选择 · 智能去重

1. 评分引擎 engine/scorer.py

Score = α·R + β·E + γ·C + δ·S − ε·M

默认权重 α=0.30 β=0.25 γ=0.20 δ=0.15 ε=0.10（侧边栏可调）

R — 相关性

话题匹配 40%Jaccard 系数
家人匹配 25%提及检测
时间衰减 15%e⁻⁰·⁰⁰¹ᵗ
语义近似 20%关键词重叠

E — 共情度

情绪匹配 40%标签一致性
历史有效 35%策略反馈
冲突矩阵 25%56格三级判定

C — 亲密度

关系权重 50%配偶1.0/子女0.9/...
记忆权重 30%intimacy_weight
访问热度 20%min(count/10,1)

S+M — 安全+罚分

风险词 50%病/死/钱/遗产
敏感话题 30%按 sensitivity_map 扣分
医学边界 20%药物/诊断 → S=0
罚分M已故亲属0.5/创伤0.4/近期负面0.3
